Crystal Collector

Rules:

1. You will be given a random number at the start of each game

2. The four crystals below each have a different, random point value associated with them.

3. Each click on a crystal will add its value to your score value.

4. You win the game by matching your score value to the target value.

5. Each time the game restarts, crystal and target values are reset randomly.

6. If you go over the target value, you lose! Be careful!
